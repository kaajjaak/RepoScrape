
# Repo Scrape

A GitHub repo and VisualParadigm revision scraper made to quickly copy the commit and revision numbers from the last 24 hours. There is also a quick overview of each members' total activity.

This project was made for a college project but feel free to use for whatever.



## Environment Variables

To run this project, you will need to add the following environment variables to your .env file

`PAT` [Creating a personal access token](https://docs.github.com/en/authentication/keeping-your-account-and-data-secure/creating-a-personal-access-token)

`REPO` The name of your repo


## Configuring Visual Paradigm

1. Export all revisions in the Teamwork client
2. Unzip the downloaded file
3. Move the revision.xml file to the root of this project


## Installation

Install RepoScrape with pip

```bash
  pip install -r requirements.txt
```
